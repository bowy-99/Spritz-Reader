# Spritz-Reader
A spritz-reader with ORP recognition with some customisation with variable WPM and color settings 
Can be used with EPUB files, PDF and txt files (I'll add more formats later on)
Ensure that the files are stored in this structure:

root/
├── app.py
└── static/
    └── index.html
